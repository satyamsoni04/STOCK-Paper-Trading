# 📈 Stock Trading Web App

A modern stock trading web application that allows users to track market trends, analyze stocks, and simulate trading decisions using an intuitive and responsive interface.

---

## 🚀 Features

- 📊 Real-time stock market dashboard  
- 📈 Interactive stock charts and price trends  
- 💼 Portfolio tracking and performance monitoring  
- 🔍 Stock search and filtering  
- 📉 Market performance insights  
- 📱 Fully responsive design  
- ⚡ Fast and optimized performance  

---

## 🛠️ Tech Stack

- React.js  
- TypeScript  
- Vite  
- Tailwind CSS  
- shadcn/ui  
- REST API Integration
